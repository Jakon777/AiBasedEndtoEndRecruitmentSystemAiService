===============================
AI SERVICE COMPLETE SETUP GUIDE
(GEMINI API VERSION)
===============================

This guide installs and runs the AI Hiring Service from scratch.

Requirements:
- Python 3.10+
- Docker installed
- Internet connection (Gemini API used for test generation)

------------------------------------------------------------
STEP 1: CREATE PROJECT FOLDER
------------------------------------------------------------

mkdir ai-service
cd ai-service

------------------------------------------------------------
STEP 2: CREATE VIRTUAL ENVIRONMENT
------------------------------------------------------------

python -m venv venv

Activate:

Windows:
venv\Scripts\activate

Mac/Linux:
source venv/bin/activate

------------------------------------------------------------
STEP 3: CREATE requirements.txt FILE
------------------------------------------------------------

Create a file named requirements.txt and paste:

fastapi==0.110.0
uvicorn[standard]==0.29.0
python-multipart==0.0.9
pdfplumber==0.10.3
spacy==3.7.4
sentence-transformers==2.6.1
scikit-learn==1.4.2
numpy==1.26.4
faiss-cpu==1.8.0
requests==2.31.0
python-dotenv==1.0.1
google-generativeai==0.7.2

------------------------------------------------------------
STEP 4: INSTALL PYTHON DEPENDENCIES
------------------------------------------------------------

pip install -r requirements.txt

------------------------------------------------------------
STEP 5: INSTALL SPACY LANGUAGE MODEL
------------------------------------------------------------

python -m spacy download en_core_web_sm

------------------------------------------------------------
STEP 6: SETUP GEMINI API
------------------------------------------------------------

Create a file named:

.env

Add your Gemini API key:

GEMINI_API_KEY=YOUR_API_KEY_HERE

This key will be used by the AI service to generate tests.

------------------------------------------------------------
STEP 7: VERIFY DOCKER INSTALLATION
------------------------------------------------------------

docker --version

If Docker is not installed:

https://www.docker.com/products/docker-desktop/

Make sure Docker Desktop is running.

Docker is required for secure code execution during coding test evaluation.

------------------------------------------------------------
STEP 8: PROJECT STRUCTURE
------------------------------------------------------------

ai-service/
│
├── main.py
├── .env
├── requirements.txt
│
├── core/
│   ├── resume_parser.py
│   ├── embedding_engine.py
│   ├── llm_client.py
│   ├── test_generator.py
│   ├── code_evaluator.py
│   ├── scoring_engine.py
│
├── routes/
│   ├── resume_routes.py
│   ├── embedding_routes.py
│   ├── test_routes.py
│   ├── evaluation_routes.py
│   ├── scoring_routes.py
│
├── uploads/

------------------------------------------------------------
STEP 9: RUN AI SERVICE
------------------------------------------------------------

python -m uvicorn main:app --reload
or
python -m uvicorn main:app --reload --reload-exclude venv

Server will run at:

http://127.0.0.1:8000

------------------------------------------------------------
STEP 10: OPEN API DOCUMENTATION
------------------------------------------------------------

Open browser:

http://127.0.0.1:8000/docs

Swagger UI will appear with all endpoints.

------------------------------------------------------------
ENDPOINTS AVAILABLE
------------------------------------------------------------

POST /resume/parse
POST /embedding/similarity
POST /test/generate
POST /evaluation/evaluate
POST /scoring/final

------------------------------------------------------------
IMPORTANT NOTES
------------------------------------------------------------

1) Gemini API key must be present in the .env file
2) Docker must be running before calling /evaluation/evaluate
3) spaCy model must be installed before /resume/parse
4) Embedding model loads automatically on first request

------------------------------------------------------------
OPTIONAL: FREEZE DEPENDENCIES FOR PRODUCTION
------------------------------------------------------------

pip freeze > requirements.txt

------------------------------------------------------------
CLEANUP (Optional)
------------------------------------------------------------

Add .gitignore file:

venv/
__pycache__/
uploads/
.env

------------------------------------------------------------
SYSTEM READY
------------------------------------------------------------

Your AI service now includes:

- Resume Parsing
- Embedding + Similarity
- AI Test Generation (Gemini API)
- Secure Docker Code Evaluation
- Composite Scoring Engine

Architecture:

Frontend -> Java Backend -> AI Service -> Gemini API

Lightweight architecture.
No heavy local LLM.
Cloud deployable.
